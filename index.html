Adobe AIR

Adobe AIR (formerly Adobe Integrated Runtime) is a cross-platform runtime system developed by Adobe Systems for building desktop applications and mobile applications, programmed using Adobe Flash, ActionScript and optionally Apache Flex.

Control line

Control line is a simple and light way of controlling a flying model aircraft. The aircraft is connected to the operator by a pair of lines, attached to a handle, that work the elevator of the model. This allows the model to be controlled in the pitch axis.

3D printing

In 3D printing, successive layers of material are formed under computer control to create an object.These objects can be of almost any shape or geometry, and are produced from a 3D model or other electronic data source.

HTML

HyperText Markup Language, commonly referred to as HTML, is the standard markup language used to create web pages. Along with CSS, and JavaScript, HTML is a cornerstone technology, used by most websites to create visually engaging webpages.

Ubuntu

Ubuntu is a Debian-based Linux operating system and distribution for personal computers, smartphones and network servers. It uses Unity as its default desktop environment. It is based on free software and named after the Southern African philosophy of ubuntu.

Ruby on Rails

Ruby on Rails, is a web application framework written in Ruby under MIT License. Rails providing default structures for a database, a web service, and web pages. It encourages and facilitates the use of web standards for data transfer, display and user interfacing.
